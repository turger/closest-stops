# Closest stops

## Getting started

Clone project

Add firebase:  
`npm install -g firebase-tools`

Login to firebase:  
`firebase login`

add .env file to project root:  
REACT_APP_APIKEY=  
REACT_APP_AUTHDOMAIN=  
REACT_APP_DATABASEURL=  
REACT_APP_PROJECTID=  
REACT_APP_STORAGEBUCKET=  
REACT_APP_MESSAGINGSENDERID=  
REACT_APP_GMAPS_KEY=  

`yarn`

`yarn start`

or 

`yarn build && firebase serve`

## Deploy to firebase

`yarn build-fb`
