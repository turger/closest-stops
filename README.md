# Closest stops

## Getting started

Clone project

Add firebase:  
`npm install -g firebase-tools`

Login to firebase:  
`firebase login`

add `.env` file to project root, see base from `.env.dist`.

`yarn`

`yarn dev`

## Deploy to firebase

`firebase use <project-id>`

`yarn build-fb`
