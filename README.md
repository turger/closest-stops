# Closest stops

## Getting started

Clone project

Add firebase:  
`npm install -g firebase-tools`

Login to firebase:  
`firebase login`

add .env file to project root:  
# https://console.cloud.google.com/apis/credentials
REACT_APP_GMAPS_KEY=  
# https://portal-api.digitransit.fi/profile
REACT_APP_HSL_KEY=

`yarn`

`yarn start`

## Deploy to firebase

`firebase use <project-id>`

`yarn build-fb`
