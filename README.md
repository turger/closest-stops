# Closest stops

## Getting started

Clone project

`yarn`

`yarn start`

Add firebase
`npm install -g firebase-tools`
Login to firebase
`firebase login`

add .env file to /src
REACT_APP_APIKEY=
REACT_APP_AUTHDOMAIN=
REACT_APP_DATABASEURL=
REACT_APP_PROJECTID=
REACT_APP_STORAGEBUCKET=
REACT_APP_MESSAGINGSENDERID=

## Deploy to firebase


`yarn build`
`firebase deploy`
